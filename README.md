# 🥬 Vegetable Detection System using YOLOv11

This repository contains a deep learning-based vegetable detection system. The model is trained to identify various vegetables from images using the YOLOv11 architecture.

## 🚀 Live Demo
You can access the live web application here: [Your Streamlit Link Here]

## 📌 Project Overview
- **Model:** YOLOv11 (Ultralytics)
- **Framework:** Streamlit
- **Language:** Python
- **Purpose:** Research and automated identification of local vegetables in Bangladesh.

## 📂 Dataset Information
The model is currently being developed and fine-tuned. 
- **Current Status:** Trained on an initial subset of images.
- **Target Classes:** 19 varieties of medicinal fruits and seeds/vegetables.
- **Total Dataset Size:** 3,800 images (Work in progress).
